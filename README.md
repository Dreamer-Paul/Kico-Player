# Kico-Player

一个简洁强大的网页音乐播放器。

## 使用方法

1. `Star` 本项目
2. 从这里 [下载](https://github.com/Dreamer-Paul/Kico-Player/archive/master.zip) 框架源码
3. 在你的项目中引用相关文件，参考 [文档](https://works.paugram.com/player) 快速使用
